## Описание

**«Система учёта тренировок по тяжёлой атлетике»** — это десктопное приложение для учёта личных финансов, разработанное в рамках производственной практики на предприятии. Приложение позволяет пользователям систематизировать доходы и расходы, анализировать финансовые привычки и контролировать бюджет в удобной визуальной форме.

## Установка
1. Убедитесь, что на компьютере установлена IntelliJ IDEA Community Edition 2024.2.2 или другая среда разработки с поддержкой Java

2. Скачайте файлы приложения:
- Main.java;
- LoginUI.java;
- RegisterUI.java; 
- MainUI.java;
- Database.java;
- ExercisesDialog.java; 
- Exercise.java; 
- Set.java; 
- Record.java; 
- Workout.java.

4. Скачайте драйвер базы данных:

sqlite-jdbc.jar (последняя версия)

4. Скомпилируйте приложение

5. Запустите приложение через Main.java

6. При первом запуске автоматически создастся файл базы данных TrainingDiary.db со всеми необходимыми таблицами и предустановленным списком упражнений
# Использование
«Система учёта тренировок по тяжёлой атлетике» — это мощный инструмент для контроля тренировочного процесса.
